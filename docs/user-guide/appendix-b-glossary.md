# Appendix B. Glossary

The glossary is maintained as a single source of truth in the repository root:
**[GLOSSARY.md](../../GLOSSARY.md)** — timber anatomy, joint types by contact,
joint parts, the frame-connection catalog, and the canonical parameter names
used by every joinery command.

It is deliberately not duplicated here: when the guide is built into a
single-page or PDF edition, the glossary is copied in at build time so the two
can never drift.
